Exercise 1

Your mission is to turn the following dropdown into a country-selection dropdown.

  1. Open a terminal in your project ng-advanced-workshop
  2. Run npm i -g json-server in that terminal to download the server that's going to run our sample back-end
  3. Run json-server --watch json/db.json to start our sample back-end
  4. Update the code in app/exercise1 so that our component loads its list of countries from http://localhost:3000/countries
